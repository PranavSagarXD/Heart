## About

**Heart** is a personal, experimental project born out of pure curiosity. It serves as an independent creative playground and an evolving portfolio for a design engineer. This project focuses on exploring the intersections of interactive design, fluid user experiences, and minimalist aesthetics.

The platform is designed to be an expressive space where experiments come to life, constantly pushing the boundaries of what a modern web portfolio can be.

## Features

- 🎨 **Experimental Design:** A space for showcasing unique design patterns and creative code experiments.
- 🕹️ **Interactive Experiences:** Built with focus on user interaction, motion, and fluid navigation.
- 🌫️ **Minimalist Aesthetic:** Clean, deliberate design choices aimed at highlighting the content.
- ⚡ **Performance Optimized:** Built using the Astro framework to ensure fast load times and an optimized experience.
- 💻 **Independent Craft:** Created as a reflection of individual design engineering and curiosity.
